# agents

My personal `AGENTS.md` — the operating manual AI coding agents read before working in my projects.

## What's in it

- **Default mode: discussion** — most of the time I want opinions and analysis, not code
- **Full loop for features/refactors** — `Analyse → Discuss → Plan → Code`, triggered only when I say so
- **Hard boundaries** — no touching `.env`/secrets, no git commits/pushes/PRs unless explicitly asked
- **No sycophancy** — push back when something is wrong, cite technical reasons

## Compatibility

Read by any agent that supports the [AGENTS.md](https://agents.md/) open format: Codex, Cursor, Claude Code, Aider, Gemini CLI, Jules, and others.

## Usage

Drop `AGENTS.md` into the root of any project. The agent picks it up automatically.
